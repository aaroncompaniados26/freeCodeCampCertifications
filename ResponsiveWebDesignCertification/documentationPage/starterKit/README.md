#WEBPACK

Simple steps to get you going.
---------

**Download or Pull This Repo*

**Install Node**
	https://nodejs.org/en/

**Download Atom**
	https://atom.io/

**Install all the node packages** 
On the root of this project run on your terminal
    
    npm install
    
**Update the node packages** 
    
    npm update

**Start the server**

  npm run watch
